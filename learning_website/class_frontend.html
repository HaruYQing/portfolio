完全針對初學者的前端網頁課程！

後端要處理的技術已經夠多了，為什麼還要學習前端呢？

本課程注重以「程式設計邏輯」講解前端網頁。幫助後端網頁工程師快速掌握前端技術，充分配合後端的需要（如 AJAX 及網頁 CSS 微調）設計前端網站。

好的設計師難找，後端工作中常常遇到前後端需同時處理的情況。此時，掌握基本前端技術是一大助力！



為什麼要學習前端網頁設計？

奇怪，架設網站明明就不難，Blogger、wix、weebly 點一點就架好了，為什麼要學習前端網頁設計呢？

請試著想想看，Dr. Cloud（您現在上的這個網站）到底是用 Blogger 還是用 wix 架設的呢？如果您單純只是要一個網站，而不在意網站是如何撰寫、如何運作的， 那可以參考課程「自己的個人網站自己架」。如果您有意願成為網頁前後端工程師、 甚至希望在未來進入網頁相關產業工作的人，那本課程絕對是您不可錯過的一堂基礎課。



網頁設計跟創業有什麼關係？

相較於在大企業中工作，面對的是維護複雜的既有系統或網站。創業則大多是從零開始， 使用有限的資源實際做出一個產品原型(Prototype)。而本課程的重點， 正是希望同學能從零開始撰寫網頁，同時學習發布(Publish)網站，瞭解並熟悉網站運作的概念。
